# Content Creation

Original content development and production workspace.

## What Goes Here

- **Scripts & Copy** - Written content, scripts, copy
- **Video Production** - Raw footage, projects, exports
- **Audio Production** - Podcasts, voiceovers, music
- **Graphics & Design** - Images, logos, branding assets
- **Planning** - Content calendars, briefs, outlines

## Suggested Subfolders

```
Content Creation/
├── Scripts/
├── Video/
│   ├── Raw Footage/
│   ├── Projects/
│   └── Exports/
├── Audio/
├── Graphics/
├── Planning/
└── Assets/
```

## Workflow

1. **Plan** - Create content brief and outline
2. **Create** - Produce the content
3. **Review** - Quality check and revisions
4. **Finalize** - Prepare for distribution
5. **Archive** - Move completed projects to archive

## File Naming Convention

`[Date]_[Project]_[Type]_[Version].[ext]`

Example: `2026-02-15_Tutorial-Series_Script_v2.docx`
