# Operations

Business operations and project management hub.

## What Goes Here

- **Project Management** - Timelines, milestones, roadmaps
- **SOPs** - Standard Operating Procedures
- **Resources** - Team resources and documentation
- **Administrative** - Contracts, invoices, legal docs
- **Tools & Systems** - Software documentation and access info

## Suggested Subfolders

```
Operations/
├── Projects/
├── SOPs/
├── Resources/
├── Administrative/
├── Tools/
└── Reports/
```

## Key Documents

### SOPs (Standard Operating Procedures)
Document recurring processes for consistency:
- Content creation workflow
- Publication process
- Quality control procedures
- Emergency protocols

### Project Management
- Active project folders
- Project templates
- Milestone tracking
- Resource allocation

## Best Practices

1. Keep SOPs updated and easily accessible
2. Regular project status reviews
3. Maintain centralized tool documentation
4. Archive completed projects quarterly
