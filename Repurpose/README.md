# Repurpose

Content adaptation and repurposing workflows.

## What Goes Here

- **Source Content** - Original content to be repurposed
- **Transformations** - Adapted versions for different platforms
- **Templates** - Repurposing templates and frameworks
- **Archive** - Library of repurposed content
- **Tracking** - What's been repurposed and where

## Repurposing Strategies

### From Long-Form Content
- Extract key quotes for social media
- Create highlight reels
- Convert to blog posts or articles
- Extract audio for podcast episodes

### From Short-Form Content
- Compile into collections
- Expand into long-form content
- Create themed compilations
- Transform into different formats

## Workflow

1. **Identify** - Select source content for repurposing
2. **Plan** - Determine target formats and platforms
3. **Adapt** - Create new versions optimized for each use
4. **Track** - Document what was created and where it's used
5. **Archive** - Store for future reference
