# Communication

Central hub for all team and client communication workflows.

## What Goes Here

- **Email Templates** - Standardized email formats for various purposes
- **Client Communications** - Correspondence, proposals, reports
- **Team Collaboration** - Internal memos, announcements, updates
- **Meeting Materials** - Agendas, notes, action items
- **Scripts** - Communication scripts for calls, presentations, etc.

## Suggested Subfolders

```
Communication/
├── Email Templates/
├── Client/
├── Team/
├── Meetings/
└── Scripts/
```

## Best Practices

1. Use clear, descriptive filenames with dates (YYYY-MM-DD format)
2. Archive old communications regularly
3. Keep sensitive information secure
4. Maintain version control for templates
