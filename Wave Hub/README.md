# Wave Hub

The central hub for the Tsunami Unleashed web presence.

## Structure

```
Wave Hub/
└── src/
    ├── _data/      - Data files (JSON, YAML) for site content
    ├── _includes/  - Reusable templates and components
    └── content/    - Main content files (pages, posts, etc.)
```

## Folder Purposes

### `src/_data/`
Store structured data files that can be used across the site:
- Navigation menus
- Site configuration
- Reusable content blocks
- API data or external feeds

### `src/_includes/`
Reusable template components:
- Headers and footers
- Navigation components
- Layout templates
- Partials and snippets

### `src/content/`
Main site content:
- Pages
- Blog posts
- Articles
- Documentation
